# Deloitte-Australia-Data-Analytics-Job-Simulation-on-Forage-

## 📌 Overview
This project is part of the Deloitte Australia Data Analytics Job Simulation on Forage, focused on applying data analysis and forensic technology concepts to real-world scenarios.

## 🔍 Key Tasks
- Analyzed structured datasets using Excel  
- Classified data and identified patterns and insights  
- Created an interactive Tableau dashboard  
- Derived actionable business conclusions  

## 🛠 Tools Used
- Microsoft Excel  
- Tableau  

## 💡 Key Learnings
- Data cleaning and classification  
- Insight generation from datasets  
- Data visualization and storytelling  
- Business-focused analytical thinking
